# Carniceria ERP

Sistema de gestión comercial orientado a carnicerías, diseñado para centralizar la operación diaria del negocio mediante el control de stock, ventas, compras, proveedores y análisis de rentabilidad.

Proyecto personal desarrollado como aplicación Fullstack, enfocado en la construcción de una solución real, escalable y mantenible, aplicando buenas prácticas de desarrollo, diseño modular y arquitectura cliente-servidor basada en API REST.

---

## Objetivo

Desarrollar una plataforma que permita optimizar la gestión integral de un comercio minorista, facilitando:

- Control de inventario y stock estimado
- Registro de ingresos y egresos de mercadería
- Gestión de ventas y cobranzas
- Administración de proveedores
- Seguimiento de deudores y reclamos
- Control de pérdidas
- Gestión de promociones y ofertas
- Análisis de ganancias y rendimiento operativo

---

## Funcionalidades

### Implementadas

- Sistema de autenticación y permisos

## Arquitectura

```bash
carniceria-erp/
│
├── frontend/       # Aplicación cliente
├── backend/        # API y lógica de negocio
├── docs/           # Documentación técnica (opcional)
└── README.md
```

Separación por responsabilidades:

- **Frontend:** interfaz de usuario y experiencia visual
- **Backend:** reglas de negocio y servicios
- **Base de datos:** persistencia de información
